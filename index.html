Vector Tile Base map Cache

  • Industry project sponsored by the RCMP E Division. The goal of this project is to create a base map tile cache of the province of British Columbia and it was created in ArcGIS Pro. The goal for this project is so that the RCMP can have an internal reference base map for computers that are not connected to the internet. Symbology was created for each dataset, as well as, showing each dataset at the appropriate zoom level.
  • This project uses datasets obtained from OpenStreetMaps and internal datasets from the sponsor. A model was created using model builder and python, to update the tile cache when importing updated datasets.

    Additionally, a python script was created to download the Community Maps Of Canada (CMOC) offline vector tile package on ArcGIS Online. This map is created by ESRI Canada and the script uses the Python API for ArcGIS Online to download the tile package. The CMOC map is used as an alternative base map.
    Link to the script here.

    There are mutiple ways to update the datasets, either by using the models or with a csv file. Documentation was created so that users can utilize the models/scripts and troubleshoot common problems that occur when rendering the tile cache.
    Link to a copy of the documentation here.

  • Geoprocessing tool created to update one or more datasets

Urban Heat Island - Esri App Challenge

  • Esri App Challenge is an event hosted by Esri Canada for student to create an app using Esri technology to create a solution to a problem in a week. More about the challenge here.
  • My group decided to create an app focusing on urban heat islands in the City of Vancouver due to recent heatwave events. I gathered and downloaded datasets from landsat 8 and calculated land surface temperature from that data. We created datasets of impervious surfaces, canopy cover and population density. The result is an Esri Storymap with the analysis of urban heat islands and showcase where vulnerable populations are during a heatwave. Keep in mind everything created for this project was made in a week.
